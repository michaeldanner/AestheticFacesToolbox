# AestheticFacesToolbox
Toolbox to analyse social and ethical aspects on aesthetics

## Requirements:
Tested with Python 3.6, matplotlib==3.2.2, numpy==1.19.0, Pillow==7.1.2, PyQt5==5.15.0
